Calculating the trip duration of each trip

trip_total_c$trip_duration<-as.numeric(difftime(trip_total_c$ended_at,trip_total_c$started_at, units ='mins'))

Finding the weekday at which the trip stated

trip_total_c$weekday<-weekdays(trip_total_c$started_at,abbreviate = FALSE)

Making column with month

trip_total_c$month<-format(trip_total_c$started_at,"%m")

Removing duplicate value with the same id from data frame

trip_total_c<-trip_total_c %>% 
  distinct(ride_id,.keep_all = TRUE)

Filtering all values in trip_duration which is not positive

trip_total_c<-trip_total_c %>% 
  filter(trip_duration>0)

Comparing number of casual and member rides

trip_total_c %>%

Member are using more ride than casual users. Member have 58% of ride while casual have only 41 %, which means that member are using more ride than casual users.

Average trip for both type of users

avg_duration <-trip_total_c %>% 
  group_by(member_casual) %>% 
  summarise(average_trip_time=mean(trip_duration))

Ploting graph for average trip duration for both casual and member

ggplot(avg_duration, aes(member_casual, y = average_trip_time, fill = member_casual))+
  geom_col() + 
  labs(title="Average trip duration Casual vs Member")+ scale_fill_brewer("Paired")

Average trip time for causal is higher than the member. Memeber are using ride quiet frequent but they spent less time in using.
